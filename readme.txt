== Installation ==

1. Lade das Plugin nach `/wp-content/plugins/vbc-sponsoring`, oder Installiere es direkt im Wordpress.
2. Aktiviere das Plugin unter 'Plugins' im WordPress
3. Benutze Einstellungen->Sponsoring um das Plugin zu konfigurieren
4. Öffne die Seite oder den Beitrag in dem du das Formular möchtest und füre folgendes ein [sponsoring]
